Counting Sort Algorithm

Counting Sort is a non-comparative, integer-based sorting algorithm that works by counting the occurrences of each unique element in the input array. It operates by creating a "count" array where the index represents the elements from the input, and the values at each index represent the frequency of those elements. After counting, the algorithm computes cumulative sums to determine the position of each element in the sorted output. Finally, it places the elements into the correct position in a new sorted array. Counting Sort is highly efficient for sorting integers within a known range, with a time complexity of O(n + k), where n is the number of elements and k is the range of input values.
