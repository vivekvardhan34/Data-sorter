Insertion Sort Algorithm

Insertion Sort is a simple, comparison-based sorting algorithm that builds the sorted array one element at a time. It works by picking an element from the unsorted part of the array and inserting it into its correct position in the sorted part. This is done by comparing the selected element with the elements in the sorted section, shifting the larger elements one position to the right to make space for the new element. The process repeats until all elements are sorted. Insertion Sort is efficient for small datasets or nearly sorted arrays, with a time complexity of O(nÂ²) in the worst case.
