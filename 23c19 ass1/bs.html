Bubble Sort Algorithm

Bubble Sort is a simple comparison-based sorting algorithm. It repeatedly steps through a list of unsorted elements, compares adjacent elements, and swaps them if they are in the wrong order (i.e., if the left element is larger than the right one). This process is repeated for each element in the list, with the largest elements "bubbling" to the end of the list in each pass. The algorithm continues until no more swaps are needed, meaning the list is fully sorted.
